!ss

It is a command used to setup channels, roles, screenshot type, total screen shots required and page name for screenshot verification process. it can only be used by users with administrator permisions. After setting up all the things, the bot will direct you on what should be done to verify screenshots.

!slot

It is a command used to setup slot system in a server. A server can only have one slot system and needs to be upgraded to premimum version of the bot for setting up more than one slot system. it can only be used by users with administrator permissions.

!ssreconfig

This command is used only by administrators to reset the existing setup, clearing previous configurations, and redesign the system from scratch. Only one screenshot verification system can exist per server, and making changes requires administrator permissions.

!emptyslot

This command is a powerful administrative tool that enables administrators to clear or reset all data, content, or items in the existing slot. Restricted to administrators, it ensures controlled use to prevent unauthorized or accidental actions by regular users.

!slotreconfig

This command is used to reconfigure the slot system in a server. It clears the existing setup, allowing administrators to start fresh and redesign the system as needed. Only users with administrative permissions can execute this command.
